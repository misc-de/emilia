Emilia

GTK4/libadwaita-Musik- und Hörspielplayer (Phosh). Flatpak-Repo für x86_64 und aarch64, signiert.

Installieren

flatpak remote-add --if-not-exists emilia https://misc-de.github.io/emilia/de.cais.Emilia.flatpakrepo
flatpak install emilia de.cais.Emilia
flatpak run de.cais.Emilia

Updates danach mit flatpak update. Quellcode: github.com/misc-de/emilia.
